## ZSH мелочи, плагины

### timer
показывает время выполнения команды
```sh
~/.zshrc
TIMER_THRESHOLD=10
# после какой задержки показывать чтобы не спамить
plugins=(... timer ...)
```

### подключение алиасов
аналогично подключать файлы в алиасы
```sh
~/.zshrc
[ -f ~/.aliases ] && source ~/.aliases
```
